# Table Definitions

## students

Stores student records for event registration and future personalization.

## events

Stores student-life events that may be attached to a campus building.

## registrations

Join table between students and events.

## buildings

Stores named campus locations for map and directory lookup.

## parking_lots

Stores campus parking locations and permit categories.

## resource_links

Stores official Purdue Northwest resource links used by backend lookup and redirect behavior.
